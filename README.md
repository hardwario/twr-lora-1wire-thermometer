## SENSOR MODULE CONNECTION


### Sensor Module R1.1 - 5 pin connector
| A | GND | VCC | GND/C | B |
|---|-----|-----|-------|---| 
| - | GND | VCC |   -   | DATA |


### Sensor Module R1.0 - 4 pin connector

| A   | GND | GND | B    |
|-----|-----|-----|------| 
| VCC | GND |  -  | DATA |


## DS18B20 sensor pinout

| VCC   | GND | DATA | 
|-----|-----|-----| 
| red | black | yellow |
| red | black | white |
